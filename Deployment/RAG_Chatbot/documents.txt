FastAPI is a web framework for building APIs.FAISS is a library for efficient similarity search.Docker helps in packaging and deploying apps.LangChain helps build LLM-based pipelines.PyTorch is used for training deep learning models.Transformers are neural networks designed for sequence-to-sequence tasks.Retrieval-Augmented Generation improves LLM answers by using external knowledge.Uvicorn is an ASGI server commonly used with FastAPI.Sentence Transformers generate vector embeddings for text.Tokenization is the process of splitting text into smaller units for models.RAG combines document retrieval with generative models like BERT or GPT.GitHub is a platform for hosting and sharing code repositories.Ngrok creates secure tunnels to localhost for public access.Colab is a cloud-based notebook environment for Python and machine learning.Pandas is a Python library for working with tabular data.Numpy provides support for arrays and numerical operations in Python.Machine learning models can be supervised, unsupervised, or reinforcement-based.The CPU version of FAISS is sufficient for small datasets and local testing.Dockerfiles define how to build a container image from source code.APIs allow software systems to communicate with each other over HTTP.
